# 🚗 Car Management System (Flask + SQLite)

A simple car management web app built with Flask. This project supports:

- Add a car 🚘
- View all cars 📋
- Update a car ✏️
- Delete a car 🗑️

## 📁 Technologies Used

- Python 3
- Flask
- SQLite (for lightweight database)
- HTML (for form interface)
